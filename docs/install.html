Install

linux cron:

If you are using crontab on your system, you can follow these steps to configure a cron job to keep your domain updated with the curl tool:

First, make a directory in your home and create a script:

mkdir kgbdns
cd kgbdns
touch kgb.sh

Now, put the following command with your credentials into the kgb.sh file:

echo url="https://www.kgbdns.com/update?domain=exampledomain&token=exampletoken&ip=" | curl -k -o ~/kgbdns/kgb.log -K -

After you save the file, you should make the script executable:

chmod 700 kgb.sh

Now, we are going to add a cron job to update the domain every 5 minutes. First, type the following to open crontab config:

crontab -e

Copy and paste the following at the bottom of the config:

*/5 * * * * ~/kgbdns/kgb.sh >/dev/null 2>&1

Okay, we are almost done. Let's head back and test the script:

./kgb.sh

You should be able to see the output of the script by typing:

cat kgb.log

If you see an OK you are good to go!
