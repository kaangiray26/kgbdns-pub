updating a domain:

To update a domain you need to make a basic HTTPS GET request to our endpoint /update with the following parameters:

* domain: <domain name> REQUIRED
* token: <user token> REQUIRED
* ip: <target ip> OPTIONAL

Here's an example:

https://www.kgbdns.com/update?domain=YOURDOMAIN&token=YOURTOKEN&ip=YOURIP

If the request was successful, you would get:

OK

Otherwise, you would get:

KO

removing a domain:

To remove a domain you need to make a basic HTTPS DELETE request to our endpoint /remove with the following parameters:

* domain: <domain name> REQUIRED
* token: <user token> REQUIRED

Here's an example:

https://www.kgbdns.com/remove?domain=YOURDOMAIN&token=YOURTOKEN

If the request was successful, you would get:

OK

Otherwise, you would get:

KO

domain limits:

Each user can create at most 5 subdomains and this number can not be increased. There are no subscriptions or paid features.
